// Background

Who I Am

My name is Michael LaCour, but online I go by Logic. I am 14 years old and have been writing Luau for four years. I got into development because I wanted to build things that actually worked, not just looked good in a screenshot.

Most of what I do lives inside Roblox Studio: game logic, data persistence, UI systems, physics, camera work, and anything else a project needs. I do not hand off work to anyone else. If it is in the portfolio, I wrote it and tested it myself.

Outside of Roblox, I build browser-based tools in plain HTML, CSS, and JavaScript. No frameworks, no dependencies, just clean code that runs anywhere. I also know Python and use it when a project calls for it.

I take freelance and collaboration work seriously. If you have a Roblox project that needs real systems, get in touch.
